# Heritage Policy Effectiveness Classifier

Нейросетевая классификация эффективности мер сохранения объектов культурного наследия по данным экспертных опросов.

## Задача

Эксперты в области архитектуры, девелопмента и охраны культурного наследия оценивают применяемые меры (ГЧП, приватизация и др.) по нескольким критериям. Модель предсказывает, к какому классу эффективности относится каждая мера в конкретном регионе.

## Классы эффективности

| Код | Метка               |
|-----|---------------------|
| 0   | Неэффективно        |
| 1   | Частично эффективно |
| 2   | Эффективно          |
| 3   | Очень эффективно    |

## Пайплайн

```
Опрос (Excel)
    │
    ▼
1. Нормализация структуры
   Каждая строка = эксперт × мера × регион
    │
    ▼
2. Кодирование категорий
   «Сильно выражено» → 3, «Слабо выражено» → 1, ...
    │
    ▼
3. Заполнение пропусков
   «Затрудняюсь ответить» → среднее по мере и региону
    │
    ▼
4. Обучение нейросети
   Dense(64, relu) → Dense(64, relu) → Dense(4, softmax)
    │
    ▼
5. Тестирование сценариев + визуализация
```

## Архитектура модели

```
Входной слой:   Dense(64, activation='relu', input_shape=(20,))
Скрытый слой:   Dense(64, activation='relu')
Выходной слой:  Dense(4,  activation='softmax')

Функция потерь: categorical_crossentropy
Оптимизатор:    Adam
Метрика:        accuracy
```

## Пример вывода

Модель возвращает вероятности принадлежности к каждому классу:

```
[[12.9%  73.3%  13.1%   0.7%]   # Частично эффективно
 [29.4%  40.9%  23.4%   6.3%]   # Частично эффективно
 [16.3%  71.6%  10.4%   1.7%]]  # Частично эффективно
```

Результаты визуализируются в виде линейных графиков с подписями мер по регионам.

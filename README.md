# Атрибутивная модель выбора мер сохранения ОКН с учетом регионального профиля

Репозиторий содержит реализацию метода выбора мер сохранения неиспользуемых объектов культурного наследия (ОКН) с учетом региональной специфики. Сохранение таких объектов является важной задачей городского управления, поскольку их состояние влияет не только на сохранность исторической среды, но и на качество городской ткани, инвестиционную привлекательность территории и возможности её дальнейшего развития.

Выбор меры сохранения в каждом конкретном случае зависит от совокупности факторов: состояния объекта, доступности ресурсов, институциональной среды, качества управления, участия локальных акторов и регионального контекста в целом. По этой причине одинаковые меры в разных регионах могут давать различный результат, а сама процедура выбора требует более структурированного и сопоставимого подхода.

## Описание метода

Метод основан на сопоставлении атрибутов мер сохранения с параметрами регионального профиля и последующем расчете интегральной оценки применимости. На выходе формируется ранжирование мер для заданного регионального сценария.

Текущая версия реализует сценарную апробацию на уровне связки «мера – регион».
